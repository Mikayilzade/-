ЭПОХИ v1.1 — установка через GitHub Pages

Что изменилось:
- В меню появилась внутриигровая «Вики: что на что влияет».
- Вики подробно объясняет ресурсы, доход, местность, особенности клеток,
  улучшения, здания, технологии, разведчика, территорию и условие победы.
- Прямо указаны ограничения текущей версии: найма юнитов и рабочих пока нет.
- Сохранение от v1 совместимо и продолжит работать.

Обновление существующей игры:
1. Распакуй архив.
2. Замени в репозитории старые index.html, sw.js и остальные файлы файлами из этой папки.
3. Дождись обновления GitHub Pages.
4. Полностью закрой игру на iPhone и открой снова. При необходимости обнови страницу в Safari один раз.

Новая установка:
1. Создай отдельный репозиторий, например epohi-game.
2. Загрузи в корень репозитория все файлы из этой папки.
3. Открой Settings → Pages → Deploy from a branch → main / root → Save.
4. Открой выданную ссылку в Safari.
5. Нажми «Поделиться» → «На экран Домой».
6. После первого полного открытия игра и сохранение работают без интернета.

Важно: не переименовывай index.html, sw.js, manifest.webmanifest и файлы иконок.
