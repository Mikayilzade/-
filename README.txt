ЭПОХИ — установка через GitHub Pages

1. Создай отдельный репозиторий, например epohi-game, чтобы не заменять 2048.
2. Загрузи в корень репозитория все файлы из этой папки.
3. Открой Settings → Pages → Deploy from a branch → main / root → Save.
4. Открой выданную ссылку в Safari.
5. Нажми «Поделиться» → «На экран Домой».
6. После первого полного открытия игра и сохранение работают без интернета.

Важно: не переименовывай index.html, sw.js, manifest.webmanifest и файлы иконок.
